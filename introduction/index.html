Prerequisite: 4-Way Handshake

WPA/WPA2 uses 4-way handshake to establish session key named Pairwise Transient Key(PTK)

Prerequisite: Principle

  • Attacker collects SSID, ANounce, SNounce, AP's MAC, STA's MAC
  • Calculates PSK using SSID
  • Inputs PSK into password based key derivation function
  • 256 bits public key comes out
  • Uses attacking tools (pyrit, cowpatty)
  • Creates 256 bit PSK coresponding password
  • Creates PTK with another variable
  • This PTK is one the the packet of handshake
  • PTK is used for verifying message integrity check
  • Attacker can crack the password if it exists in dictionary

Prerequisite: Rainbow Table

  • 是一张采用各种加密算法生成的明文和密文的对照表
  • 通过使用相关软件对彩虹表进行查找、比较、运算
  • 能够迅速得出此加密字符串对应的明文
